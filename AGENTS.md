# AGENTS.md

이 문서는 Pokefolio 저장소에서 작업하는 개발자와 자동화 에이전트를 위한 기본 지침입니다.

## 작업 원칙

- 작업 범위를 `frontend`, `backend`, `docs`로 명확히 구분합니다.
- 요청되지 않은 기술 스택이나 의존성을 임의로 도입하지 않습니다.
- 기능 구현과 함께 필요한 문서를 갱신합니다.
- 비밀키, 인증 정보, 로컬 환경 파일은 저장소에 커밋하지 않습니다.
- 기존 변경 사항을 보존하고, 작업과 무관한 파일은 수정하지 않습니다.

## 디렉터리 역할

- `frontend/`: 프런트엔드 소스와 관련 설정
- `backend/`: 백엔드 소스와 관련 설정
- `docs/`: 요구사항, 아키텍처, API 및 운영 문서

## 초기화 상태

현재 프로젝트에는 구현 코드가 없습니다. 기술 스택과 세부 구조는 별도 요구사항이 정해진 뒤 추가합니다.
