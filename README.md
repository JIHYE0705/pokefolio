# Pokefolio

Your Pokémon Card Portfolio.

An AI-powered collection manager for Pokémon TCG collectors.

Features:
- 📚 Collection Management
- 📖 Opening Log
- 🗂 AI Binder Layout
- 🤝 Trade Suggestions
- ✨ Collection Insights

## 프로젝트 구조

```text
pokefolio/
├── frontend/   # 프런트엔드 애플리케이션
├── backend/    # 백엔드 애플리케이션
├── docs/       # 기획 및 기술 문서
├── README.md
├── AGENTS.md
└── .gitignore
```

## 디렉터리 안내

- `frontend/`: 사용자 화면과 클라이언트 로직을 배치합니다.
- `backend/`: API와 서버 로직을 배치합니다.
- `docs/`: 요구사항, 설계, API 명세 등 프로젝트 문서를 배치합니다.
